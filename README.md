
# Diet Calendar Planner System

## Description
A web-based system that helps users plan meals based on their height, current weight, and target weight. It automatically generates diet plans and sends email notifications for meal and activity reminders.

## Features
- User Registration & Login
- Input Height, Weight, Target Weight
- Auto-generated Meal Plans
- Calendar-based Planner
- Activity Suggestions (Jogging)
- Email Notifications
